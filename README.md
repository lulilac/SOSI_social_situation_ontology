# SOSI – Social Situation Ontology

SOSI (Social Situation Ontology) is an OWL ontology designed to represent and categorize **social situations in Human–Robot Interaction (HRI)**.

The ontology provides a structured way to describe the **context, participants, and interaction dynamics** of social situations, enabling consistent annotation, reasoning, and analysis of social interaction scenarios involving humans and robots.

The conceptual foundation of SOSI is inspired by **Kenneth Burke’s Dramatistic Pentad**, a framework from rhetoric and social theory used to analyze human actions.

---

# Conceptual Foundation: Burke’s Dramatistic Pentad

Kenneth Burke’s **Dramatistic Pentad** describes human action through five elements:

- **Act** – what is happening
- **Scene** – where and in what context the action occurs
- **Agent** – who performs the action
- **Agency** – how the action is performed
- **Purpose** – why the action occurs

SOSI adapts this framework to structure social interaction contexts relevant to **human–robot interaction research**.

Rather than modeling narrative structure, SOSI focuses on **observable characteristics of social situations**.

---

# Ontology Structure

The central concept of the ontology is:

**SocialSituation**

A `SocialSituation` represents a specific interaction context and is characterized through several dimensions that describe the interaction environment, participants, and communication patterns.

---

# Core Dimensions of SOSI

## Scene (Context)

Represents the **environment in which the interaction occurs**.

Properties include:

- **Location**
- **Physical environment type** (e.g., indoor / outdoor)
- **Situation formality**
- **Crowd context / social background**

These describe the situational context surrounding the interaction.
